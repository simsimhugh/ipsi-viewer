# 01. 요구사항

## 1. 배경

수도권 학부모는 자녀의 중학교 선택 시 두 가지 질문을 동시에 가진다.

1. 어떤 중학교가 특목·자사·영재·외고·과학·국제고로 얼마나 잘 진학시키는가?
2. 그 중학교 학군에 진입하려면 어떤 아파트를 봐야 하는가?

기존 서비스(호갱노노, 아실, 학군지도 등)는 일부 정보를 제공하지만, 공시 데이터 원본으로부터 일관된 기준으로 정렬·필터링·비교할 수 있는 도구가 부족하다.

## 2. 사용자

- 1차 사용자: 수도권 거주 중1~초6 자녀의 학부모
- 2차 사용자: 학군 분석에 관심 있는 부동산 투자자, 교육 컨설턴트

## 3. 범위

### In scope

- 지역: 서울특별시, 경기도, 인천광역시
- 학교 단계: 중학교 (졸업 후 고교 진학 결과)
- 데이터 종류: 학교알리미 "졸업생의 진로 현황" 공시 항목 + 학구도(통학구역) 폴리곤 + 아파트 단지(국토부 실거래가) 매핑

### Out of scope (현 단계)

- 초등학교·고등학교 진학 데이터
- 비수도권 지역
- 학원·내신·수능 등급 등 사교육 정보
- 부동산 거래 중개 기능
- 실시간 알림·푸시

## 4. 기능 요구사항

### FR-1 학교별 진학 결과 조회

각 중학교에 대해 다음 카테고리별 졸업생 진학 인원·비율을 보여준다.

- 일반고
- 자율형 사립고(자사고)
- 자율형 공립고
- 특목고: 외국어고 / 국제고 / 과학고 / 예술고 / 체육고 / 마이스터고
- 영재학교
- 특성화고
- 기타(대안교육·미상 등)

### FR-2 학교 랭킹 테이블 뷰

- 컬럼: 학교명, 시·군·구, 졸업생 총원, 각 카테고리 인원·비율, 컴포지트 점수("특목+영재+자사 합산 %" 등 사용자 선택)
- 정렬: 모든 수치 컬럼에서 오름·내림차순
- 필터: 시·군·구, 학교명 검색, 비율 임계값

### FR-3 학교 상세 대시보드

- 연도별 진학 추이 차트(가능한 연도 범위 내)
- 해당 학교의 학구 폴리곤 내 아파트 단지 목록 (단지명, 도로명주소, 최근 실거래가)
- 공시 일자 라벨 표시

### FR-4 단지 → 학교 역조회

- 아파트 단지명 또는 도로명주소로 검색 → 배정 중학교 확인

### FR-5 새로고침

- 학교 단위: 사용자가 [새로고침] 클릭 → 학교알리미 페이지 재스크래핑 트리거. 같은 학교 6시간 쿨다운.
- 단지 단위: 실거래가 최신 데이터 강제 재페치.

### FR-6 공시 시점 표시

- 모든 진학 데이터 카드에 "공시일: YYYY-MM-DD" 형식 라벨 표기

## 5. 비기능 요구사항

### NFR-1 갱신 주기

| 데이터 | 자동 cron | 사용자 새로고침 |
|---|---|---|
| 학교 진학 결과 | 일 1회 변경감지 스크래핑 | 학교당 6시간 쿨다운 |
| 학구도 폴리곤 | 월 1회 체크 | 의미 없음 |
| 아파트·실거래가 | 일 1회 동기화 | 단지당 1시간 쿨다운 |
| 단지-학교 매핑 | 폴리곤/단지 변경 시 재계산 | — |

### NFR-2 응답 속도

- 메인 테이블 첫 페인트 < 2초 (수도권 약 1500개 학교)
- 학교 상세 대시보드 < 1.5초

### NFR-3 비용

- 월 운영비 USD 20 이하를 목표 (Firebase 무료 한도 우선 활용)

### NFR-4 데이터 정합성

- 학교알리미 원본과의 마지막 동기화 시각, 다음 갱신 예정 시각을 모든 학교 카드에 노출

### NFR-5 법적·윤리적

- robots.txt 준수, 트래픽 분산(스크래퍼 동시성 제한, 학교당 일 1회 한도)
- 공공데이터 라이선스(공공누리 제3유형 등) 표기
- 학교알리미·NEIS·schoolzone.emac.kr·국토부 실거래가 모두 공공 출처 — 출처 명시

## 6. 제약사항

- 학교알리미는 공식 OpenAPI에서 "졸업생 진로 현황"을 제공하지 않아 Playwright headless 브라우저 스크래핑이 유일한 경로
- 학구도 SHP는 연 2회만 갱신, 그 사이 변경된 학구는 반영되지 않을 수 있음 (사용자 안내 필요)
- 호스트 OS(Ubuntu 26.04)와 Playwright 공식 지원 OS(20.04/22.04/24.04) 간 불일치 — 회피 방안은 [05-implementation-plan.md](05-implementation-plan.md) 참고
